Introduction

In addition to normal slots, you may also want to use the Illusionary slots.
Using Twilight Runes you can increase the amount of Illusionary pockets.
Illusionary pockets can be used instead of your main pockets.


Feeling the need to use illusionary pocket, you simply click on it.
It will move the potion or scroll to the main slot.
Now it can be used like if it was always there.

However, once the main or corresponding to it illusionary slot is used then the other one cannot be used.
As shown on the picture above:
On the left side you can see the main and illusionary pocket,
but then the scroll from main pocket was used, so illusionary one is no longer available, as seen on the right.

Illusionary slots are added by using Special runes.
The runes are not being placed in armor and they will not affect your belts.

Obtaining and Use of Twilight Runes

The runes can be purchased:
- After reaching enough Shadow Catchers Reputation.
- After reaching enough Reputation Rating points.
- Received during special events, for example Valuable Chests.

There is no specific order in which the runes should be used.
However, grey, green, blue and purple runes can only be used one time.
Red runes can add overall 4 more illusionary slots, but each time they will require double the amount:
- 1 Red Rune for 1st illusionary slot
- 2 Red Runes for 2nd illusionary slot
- 4 Red Runes for 3rd illusionary slot
- 8 Red Runes for 4th illusionary slot
Overall for all 4 illusionary slots from them you will need to use 15 Red Runes.
